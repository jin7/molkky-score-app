# Mölkky Score Tracking

This is a web application for tracking scores in the game Mölkky.

## Features

- Add and remove players
- Input scores for each player
- Display the current game status and leaderboard
- Reset the game

## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/your_username_/Project-Name.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the app
   ```sh
   npm start
   ```
